Uncle小说安卓版是一款能够对全网小说进行解析与下载的工具，同时支持文本小说与有声小说。主要包含功能有：

- 【规则解析】支持用户自定义爬虫规则进行下载小说，包含搜索、目录、正文。
- 【自动解析】核心功能，无需规则，只需要任意网站的任意小说目录，即可解析下载
- 【小说下载】文本小说支持EPUB与TXT格式
- 【任务管理】支持多线程、断点续传、失败重试，可自行配置。
